## Base URL

BooFi Engine API is built on REST principles and is served over HTTPS. To ensure data privacy, unencrypted HTTP is not supported.

The Base URL for all API endpoints is:

```bash Terminal
https://engine-api.boofi.ai
```

## Authentication

Authentication to BooFi Engine API is performed via the Authorization header with a Bearer token. To authenticate, you need to include the Authorization header with the word `Bearer` followed by your token in your API requests like so:

```bash Terminal
Authorization: Bearer <Your-Token>
```
